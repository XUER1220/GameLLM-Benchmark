## 任务身份
请使用 Python 3 和 pygame 实现单文件游戏 `Polarity Fleet`。它是阵列射击游戏，但不是标准 Space Invaders 复刻。玩家必须切换弹药极性，只有颜色匹配的子弹才能消灭目标。

## 坐标与时间
1. 窗口固定 `800x600`，标题固定 `Polarity Fleet`，帧率固定 `60 FPS`。
2. 战斗区域固定为 `pygame.Rect(40,80,720,480)`。所有碰撞使用整数 `pygame.Rect`。

## 颜色表
`BACKGROUND=(7,13,26)`，`FIELD=(14,24,42)`，`PLAYER=(236,242,255)`，`CYAN=(62,218,255)`，`AMBER=(255,182,72)`，`ENEMY_BULLET=(255,92,122)`，`TEXT=(238,244,255)`，`DANGER=(255,86,108)`，`WIN=(126,244,158)`。

## 布局与实体
1. 玩家矩形固定为 `pygame.Rect(370,524,60,28)`，水平速度固定 `6` 像素/帧，限制在 `x=40..700`。
2. 玩家子弹固定 `6x16`，速度固定 `(0,-8)`。按下发射时，子弹水平居中于玩家，底部等于玩家顶部。屏幕内最多存在 `3` 发玩家子弹。
3. 敌人固定 `3` 行 x `7` 列，共 `21` 个。每个敌人 `38x24`，初始左上角公式为 `(132 + col*78, 130 + row*54)`。
4. 敌人极性按 `(row+col)%2` 固定：偶数为青色，奇数为琥珀色。
5. 敌方子弹固定 `6x14`，速度 `(0,5)`。敌方射击帧固定为每隔 `90` 帧一次；射手索引序列固定循环 `[0,6,3,10,15,5,18,8]`，每次从当前存活敌人按从左到右、从上到下排序后用 `索引 % 存活数` 选择。

## 输入
1. 持续按左/右方向键移动。按 `SPACE` 发射。按 `X` 在青色和琥珀色弹药之间切换。
2. 初始弹药极性固定为青色。射击冷却固定 `220 ms`。
3. `R` 重开，`ESC` 或关闭窗口退出。

## 更新顺序
每帧严格执行：处理事件 -> 更新玩家 -> 更新敌人阵列 -> 生成敌方子弹 -> 更新全部子弹 -> 处理玩家子弹与敌人 -> 处理敌方子弹与玩家 -> 移除越界子弹 -> 检查胜负 -> 绘制。终局后冻结。

## 独特规则
1. 玩家弹药极性在青色和琥珀色之间切换，初始弹药极性固定为青色。
2. 玩家子弹只有碰到同极性敌人时才能消灭目标；异极性命中只移除子弹并保留敌人。
3. 敌阵整体反向和下移使用固定边界，敌方射击使用固定射手序列，禁止随机射击。

## 计分与终止
1. 玩家子弹碰到同极性敌人：敌人消失，子弹消失，分数 `+12`。
2. 玩家子弹碰到异极性敌人：敌人保留，子弹消失，分数 `-2`，但分数不能低于 `0`。
3. 敌方子弹碰到玩家：子弹消失，生命 `-1`，玩家重置到初始位置。玩家总生命固定 `3`。
4. 全部敌人被消灭则胜利；生命为 `0` 或任一敌人底部大于等于玩家顶部则失败。

## 绘制顺序与 HUD
1. 绘制背景、战斗区域、敌人、玩家子弹、敌方子弹、玩家、HUD、终局文字。
2. HUD 字号 `27`：`Polarity Fleet` 于 `(20,20)`，`Score: <n>` 于 `(390,20)`，`Lives: <n>` 于 `(540,20)`，`Ammo: CYAN|AMBER` 于 `(650,20)`。
3. 终局字号 `48` 于 `(285,250)` 绘制 `You Win` 或 `Game Over`，字号 `28` 于 `(285,315)` 绘制 `Press R to Restart`。

## 禁止项
禁止掩体、Boss、升级、道具、多关卡、随机射击、音效、素材和忽略极性匹配的经典射击逻辑。

## 提交前检查
确认双色敌阵、`X` 切换、异色扣分、固定射手序列、阵列反向、生命、HUD、冻结、`R` 和 `ESC` 均已实现。
