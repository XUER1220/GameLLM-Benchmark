## 任务身份
请使用 Python 3 和 pygame 实现单文件游戏 `Relay Defense`。它是固定路径塔防游戏，但不是标准 Tower Defense 模板。敌人具有双色护盾，三种塔对护盾有明确克制；塔只能建在指定建造格。

## 坐标与时间
1. 窗口固定 `800x600`，标题固定 `Relay Defense`，帧率固定 `60 FPS`。
2. 地图固定为 `20` 列 x `15` 行，单格 `32x32`。地图矩形固定 `(0,80,640,480)`，右侧面板固定 `(640,80,160,480)`。
3. 网格 `(col,row)` 的中心像素为 `(col*32+16,80+row*32+16)`。敌人使用浮点中心位置，绘制和碰撞时转为整数。
4. 时间全部使用毫秒。

## 颜色表
`BACKGROUND=(10,15,26)`，`GROUND=(32,48,58)`，`GRID=(52,70,78)`，`PATH=(166,132,92)`，`ENTRANCE=(92,218,255)`，`BASE=(255,104,116)`，`PANEL=(22,30,46)`，`PAD=(74,94,112)`，`ARROW=(96,224,142)`，`CANNON=(255,174,76)`，`SLOW=(118,170,255)`，`CYAN_SHIELD=(72,222,255)`，`AMBER_SHIELD=(255,190,76)`，`NORMAL=(232,238,248)`，`FAST=(255,126,154)`，`ARMOR=(176,142,255)`，`PROJECTILE=(250,250,255)`，`TEXT=(238,244,255)`。

## 布局与实体
1. 敌人路径折点网格固定为 `[(0,7),(4,7),(4,3),(9,3),(9,11),(14,11),(14,5),(19,5)]`。敌人必须沿折线中心连续移动。入口为 `(0,7)`，基地为 `(19,5)`。
2. 可建造格固定为 `[(1,5),(2,9),(5,5),(6,2),(6,8),(7,12),(10,1),(10,6),(11,10),(12,13),(15,3),(15,8),(16,12),(17,4),(18,8)]`。只能在这些格子建塔，每格最多一座。
3. 三种塔基础数据固定为：
`Arrow Tower: key=1, cost=50, range=112, damage=8, cooldown=700`
`Cannon Tower: key=2, cost=80, range=104, damage=14, cooldown=1100, splash_radius=44`
`Slow Tower: key=3, cost=70, range=108, damage=4, cooldown=900, slow_duration=1800`
4. 塔升级最多 `2` 次。Arrow、Cannon、Slow 每次升级价格分别固定为 `35`、`56`、`49`。等级从 `0` 开始；每升一级，射程 `+12`，伤害变为 `base_damage * (1 + 0.4*level)`。
5. 三类敌人固定为：
`normal: hp=40, speed=1.2, reward=12, radius=11`
`fast: hp=28, speed=2.0, reward=14, radius=9`
`armor: hp=80, speed=0.8, reward=20, radius=13`
6. 五波固定为：
`wave1=["normal"]*6`
`wave2=["normal"]*5+["fast"]*3`
`wave3=["normal"]*4+["armor"]*3+["fast"]*4`
`wave4=["armor"]*4+["fast"]*6+["normal"]*4`
`wave5=["armor"]*6+["fast"]*8+["normal"]*5`
7. 每波敌人按列表顺序每隔 `650 ms` 生成。每波开始前固定准备 `3000 ms`。敌人护盾按全局生成序号交替：偶数为青色，奇数为琥珀色。
8. 初始金币固定 `180`，基地生命固定 `20`。

## 输入
1. 按 `1/2/3` 选择塔型。鼠标左键点击空的指定建造格时，金币足够才建造。鼠标右键点击已有塔时，金币足够且等级小于 `2` 才升级。
2. `R` 重开，`ESC` 或关闭窗口退出。

## 更新顺序
每帧严格执行：处理事件 -> 更新准备倒计时和生成计时 -> 生成敌人 -> 沿路径移动敌人 -> 处理到达基地 -> 按建造顺序更新塔 -> 生成弹道 -> 立即结算命中效果 -> 移除死亡敌人并发奖励 -> 推进波次 -> 检查胜负 -> 绘制。终局后冻结。

## 独特规则
1. 塔在射程内选择路径进度最大的敌人；相同进度按生成序号最小者。禁止随机选择。
2. Arrow Tower 对青色护盾伤害倍率 `2.0`，对琥珀色护盾倍率 `0.5`。
3. Cannon Tower 对琥珀色护盾伤害倍率 `2.0`，对青色护盾倍率 `0.5`。命中后对半径 `44` 内全部敌人按各自护盾计算伤害。
4. Slow Tower 伤害倍率固定 `1.0`，命中目标后使其速度变为基础速度的 `50%`，持续 `1800 ms`，并立即将目标护盾在青色与琥珀色之间切换。

## 计分与终止
1. 敌人到达基地时基地生命 `-1`，该敌人消失且不给奖励。
2. 敌人 HP 小于等于 `0` 时消失并增加对应奖励金币。
3. 五波全部生成完毕且场上无敌人、基地生命大于 `0` 时胜利。基地生命小于等于 `0` 时失败。
4. 终局后冻结，只保留事件处理、绘制、`R` 和 `ESC`。

## 绘制顺序与 HUD
1. 绘制背景、地图地面、网格、宽 `24` 像素路径折线、入口、基地、建造格、塔、射程圆、敌人、护盾圆环、弹道、右侧面板、HUD、终局文字。
2. 塔为格中心半径 `11` 圆；敌人按类型半径绘制；护盾为敌人半径外加 `3` 的宽 `2` 圆环；弹道为宽 `2` 直线，保留 `120 ms`。
3. 鼠标悬停已有塔或空建造格时绘制该塔型射程圆，圆环宽 `1`。塔中心绘制等级数字 `0/1/2`。
4. 顶部 HUD 字号 `24`：`Relay Defense` 于 `(16,20)`，`Base: <n>` 于 `(210,20)`，`Gold: <n>` 于 `(330,20)`，`Wave: <n>/5` 于 `(450,20)`，`Enemies: <n>` 于 `(570,20)`。
5. 右侧面板字号 `22`：`Selected` 于 `(660,110)`，当前塔名于 `(660,140)`，三种塔快捷键和价格从 `(660,200)` 起每行间隔 `38`，准备倒计时于 `(660,360)`。
6. 终局字号 `48` 于 `(210,270)` 绘制 `You Win` 或 `Game Over`，字号 `28` 于 `(220,330)` 绘制 `Press R to Restart`。

## 禁止项
禁止非指定格建塔、随机路径、随机波次、随机目标、拖拽编辑、技能卡、联网、音效、素材和忽略护盾倍率的经典塔防实现。

## 提交前检查
确认路径折点、十五个建造格、三种塔、升级价格、五波列表、三类敌人、双色护盾、克制倍率、Slow 切盾、目标选择、HUD、冻结、`R` 和 `ESC` 均已实现。
